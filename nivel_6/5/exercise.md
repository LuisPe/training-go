#### Ejercicio #5

1. Crea un tipo cuadrado.
2. Crea un tipo círculo.
3. Adjunta un método a cada unos de los tipos creado que calcule y retorne el área
    ```
    Área de un círculo= π r 2
    Área de un cuadrado= b * h
    ```
4. Crea un tipo forma que defina una interfaz como cualquier cosa que tenga el método área
5. Crea una func info el cual toma un tipo forma e imprime el área de la misma.
6. Crea un valor de tipo cuadrado.
7. Crea un valor de tipo círculo.
8. Usa la func info para imprimir el área de un cuadrado.
9. Usa la func info para imprimir el área de un círculo.
